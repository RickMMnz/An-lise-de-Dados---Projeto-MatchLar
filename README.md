# Projeto de Análise de Dados MatchLar

Este projeto foi desenvolvido e otimizado para ser executado **exclusivamente no Google Colab**.

## Como Utilizar o Projeto

Para garantir a execução correta do projeto, siga as instruções abaixo:

1.  **Acesso ao Google Colab**: Abra este notebook diretamente no Google Colab.

2.  **Montagem do Google Drive**: O notebook requer acesso ao Google Drive para carregar o arquivo `dados_imobiliarios.csv`. Certifique-se de que o arquivo CSV esteja no caminho esperado:
    `/content/drive/MyDrive/(pasta onde o arquivo csv estará salvo no seu drive)`.
    A primeira célula de código após o título "Montar o Drive" irá solicitar a autenticação para montar seu Drive.

3.  **Execução das Células**: Execute as células do notebook em sequência, desde a primeira até a última. É crucial respeitar a ordem, pois as células dependem de variáveis e processamentos definidos nas etapas anteriores.

    *   **Opção 1: Download do Notebook**: Você pode fazer o download deste arquivo (`.ipynb`) e abri-lo diretamente no Google Colab.
    *   **Opção 2: Copiar e Colar**: Se preferir, você pode copiar o conteúdo das células (tanto de texto quanto de código) e colar em um novo notebook do Google Colab. Lembre-se de manter a distinção entre células de texto (Markdown) e células de código (Python).

## Recursos Adicionais no GitHub

No repositório do GitHub, você encontrará:

*   **`dados_imobiliarios.csv`**: O conjunto de dados utilizado neste projeto.
*   **`criacao_dados`**: O script Python que foi utilizado para gerar o arquivo `dados_imobiliarios.csv`.
*   **`analise_dados`**: O script python utilizado para a realização das análises dos dados do projeto.

## Importante

Qualquer problema de sincronização de arquivos ou incompatibilidade de ambiente pode ser evitado executando o projeto 100% dentro do ambiente Google Colab, conforme as instruções acima.
